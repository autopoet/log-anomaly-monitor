# Log Anomaly Monitor

A lightweight distributed log collection, anomaly detection, and real-time monitoring demo built for a course project.

Planned stack:

- Python
- RabbitMQ
- FastAPI
- WebSocket
- SQLite
- ECharts

The project will simulate multiple log collection nodes, publish structured log events to a message queue, analyze recent logs per device, detect severe error spikes, and display real-time device status in a web dashboard.
